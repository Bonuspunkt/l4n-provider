# l4n provider

## usage
install the provider
``` sh
npm i l4n-provider
```
install your supported games ex.
``` sh
npm i l4n-provider-csgo
```

## setup
```
npx l4n-provider init
```
check the created `settings.js` file

## launch
``` sh
node ./index.js
```
